Introdução a iframes em websites


Para o desenvolvedor que trabalha principalmente com Webdesign e programação é quase imprescindível entender o conceito de iframes. Atualmente
é comum quando se trabalha construindo a estrutura de alguns sites, se introduzir imagens ou tabelas, para deixar o conteúdo mais dinâmico para os
usuários, algo que não era pensado inicialmente nos primeiros sites. Portanto, foram necessárias serem criadas tags para inserirem imagens, vídeos
e até páginas HTML, estas últimas sendo introduzidas através das tags Frameset e a tag Iframe.

A tag Frameset

Segundo a página do Acervo Lima:

"A tag <frameset> em HTML é usada para definir o conjunto de quadros. O elemento <frameset> contém um ou mais elementos de quadro. É usado para especificar o número de linhas e colunas no conjunto de quadros com seus pixels de espaços. Cada elemento pode conter um documento separado."
Na versão atual do HTML, ou seja, o HTML5, a tag <frameset> não é compatível.

A tag Iframe

Diferente da tag <Frame>, a tag <Iframe> trata-se de uma tag onde é possível anexar páginas HTML completas como frames em um website, como explica a página do MDN.

"O elemento HTML <iframe> (ou elemento HTML inline frame) representa um contexto de navegação aninhado, efetivamente incorporando outra página HTML para a página atual. Em HTML 4.01, um documento pode conter uma cabeça e um corpo ou uma cabeça e um conjunto de quadros, mas não tanto um corpo e um conjunto de quadros. No entanto, um <iframe> pode ser usado dentro de um corpo de documento normal."
