Exemplo de <Iframe>

A tag <Iframe> permite integrar qualquer tipo de arquivo HTML dentro de uma página web. O que é fundamental para qualquer programador que queira trabalhar cm Front-end(onde se é necessário usar HTML, CSS e JavaScript).
